注册安全工程师网络教育平台学员服务条款

1、服务条款确认和接纳

注册安全工程师网络教育平台(以下称“平台”、“本平台”或“本网站”)为济南晋安注册安全工程师事务所有限公司（以下简称“济南晋安”）所属的注册安全工程师提供网络继续教育及考前培训的网站，本平台所提供的产品及服务将完全按照其发布的章程、条款和规则执行。用户在平台完成注册（报名）程序成为平台用户，视为完全了解并接受本服务条款的全部内容。

用户接受本服务条款视为同意接收本平台通过电子邮件、客户端、网页、短信、微信或其他合法方式，不定期发送与学习、考试资讯的相关信息。

本平台有权根据实际情况随时修改本服务条款，并在相关网站页面上提示修改内容。学员若继续使用平台所提供的网络服务，视为同意并接受变更后的服务条款。

2、学员信息

为保障注册机构用户能及时、全面的享受平台所提供的各项网络服务，机构用户应：

2.1 按照注册页面的提示填写准确、真实、详尽的机构资料；

2.2 若上述信息发生变化，请立即进行更新，保持机构资料的准确性。

若学员提供信息不真实、不完整，平台可根据具体情况保留随时终止向上述学员提供网络服务的权利。

本平台承诺对机构用户隐私严格保密，不以任何形式公开、透露注册机构的名称、地址、电子邮箱、联系人、联系方式等信息，但以下情况除外：

2.2.1 机构用户同意并授权平台公开或透露上述信息；

2.2.2 为提供用户所要求的产品和服务，而必须和第三方分享用户的个人信息；

2.2.3 法律法规或有关国家权力机关要求平台提供学员的个人信息；

2.2.4 其他本平台根据法律或者本平台政策认为合适的披露。

3、机构用户代码、密码及使用限制

机构用户注册（报名）成功，成为平台的用户，其所成功注册的账号为平台识别用户身份并借以提供产品及服务的唯一标识。机构用户应妥善管理其账号，不定期更换密码，确保机构账户的安全并对机构用户账号下的行为、操作承担全部责任。

对于机构用户因自身原因造成的学员代码遗失、密码泄漏所造成的各种损失（可能），以及学员在其机构账号下所进行的一切操作，平台不承担任何责任。但机构用户若发现其账号被盗用或存在非其控制的行为或操作，应立即通知平台。 学员账号供且仅供学员个人使用，学员不得以任何方式与其他第三方共享学员账号；若学员报名平台课程，不得通过学习账号公开学习课程（包括但不限于向任何第三方透露学习课程、与他人共享学员账号、将自己的学员账号提供给第三方使用、将学习课程公开播放或以任何方式供多人同时使用）。

上述情况一经发现，平台有权立即关闭相应学员账号的学习权限，学员已支付的课程费用不予退还，同时平台会进一步追究学员违规使用学员账号所引起的法律责任。

4、学员的缴费与在线课程的学习

机构在注册完成后，机构用户应为其报名的培训课程所学习的网络课程进行付费，如相关学员直接支付所学的网络课程费用则视同本机构已支付该费用。

付费方式分第三方支付平台线上支付与线下支付两种形式；

4.1 第三方支付平台线上支付主要包括支付宝支付与微信支付，通过第三方支付平台线上支付的，在支付完成后所对应的网络课程缴费状态即时确认；

4.2 线下支付主要网银转账、银行柜台汇款、银行柜台转账、第三方支付平台向银行账户转账等形式，包括支付宝支付与微信支付，通过线下支付完成后所对应的网络课程缴费状态三个工作日内确认。

( 2 ) 损害国家荣誉和利益的；

( 3 ) 煽动民族仇恨、民族歧视、破坏民族团结的；

( 4 ) 破坏国家宗教政策，宣扬邪教和封建迷信的；

( 7 ) 煽动非法集会、结社、游行、示威、聚众扰乱社会秩序的；

( 8 ) 其他违反法律、法规、规章、条例以及社会道德规范所限制或禁止的内容的。
